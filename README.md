Fazendinha 🌱

Um jogo simples de fazenda feito em Python usando Tkinter, com sistema de plantio, crescimento de culturas, inventário, salvamento de jogo e ciclo de dias.

Baseado no arquivo principal do projeto

📸 Funcionalidades
Menu principal
Criar e continuar jogos salvos
Sistema de inventário
Plantio de alface
Crescimento de culturas com tempo
Regador com água limitada
Lago para reabastecimento
Colheita
Sistema de dias e horário
Salvamento automático/manual
Movimentação do jogador
Pause menu
Interface gráfica com Tkinter
🎮 Controles
Tecla / Ação	Função
W A S D	Movimentação
Clique esquerdo	Usar ferramenta/item
Clique direito	Colher alface
ESC	Abrir menu de pausa
🧰 Ferramentas
Enxada

Prepara o solo para plantio.

Regador

Molha a terra e pode ser recarregado no lago.

Sementes de Alface

Permitem plantar alface em solo preparado.

📦 Requisitos
Python 3.10+
Tkinter (normalmente já vem com Python)
▶️ Como executar

Clone o repositório:

git clone https://github.com/seuusuario/fazendinha.git

Entre na pasta:

cd fazendinha

Execute:

python main.py
💾 Saves

Os jogos salvos ficam automaticamente na pasta:

/saves

Os saves são armazenados em formato JSON.

🏗️ Estrutura do Projeto
fazendinha/

🌱 Sistema de Crescimento

A alface possui 3 estágios de crescimento:

Broto
Crescimento intermediário
Pronta para colheita

O crescimento acontece conforme o tempo avança no jogo.
